- Use the `regionshot` skill and the `regionshot` command on PATH as the preferred project tool for macOS screenshots, running-app discovery, desktop app/window/menu inspection, visible-window fallback capture, AX window listing/raising, in-window crops, menu capture, occluded-window capture, Accessibility-driven UX actions, app launch/quit/activation, keyboard/mouse input, clipboard, waiting, and window management.
- Prefer `regionshot --help` and `regionshot <subcommand> --help` for exact command syntax instead of guessing or using raw `screencapture`, System Events AppleScript, or generic screenshot tooling first.
- Use RegionShot app discovery when the exact app name is unknown. If ScreenCaptureKit app/window capture fails or visible pixels are enough, use visible-window listing/capture before raw rectangle capture. If Screen Recording permission blocks app/window inspection but Accessibility works, use `regionshot ax --app APP windows` and `regionshot ax --app APP raise --window-index N` before AppleScript fallbacks.
- Use RegionShot menu-bar modes for menu-bar/status-item menus or popovers from accessory/background apps; use `regionshot menu --app APP press-item TEXT` to choose child items inside status menus; use rectangle capture only for visible UI that is not exposed through app/window/visible-window/menu-bar commands.
- For local app development loops, prefer RegionShot primitives such as `regionshot launch PATH|BUNDLE_ID --wait-window`, `regionshot ax --app APP wait-for-element ...`, `set-value`, `type`, `key`, `click`, `drag`, `scroll`, `move`, `resize`, `close`, `minimize`, `regionshot doctor`, and `regionshot clipboard` before ad hoc sleeps or AppleScript.
- If System Events AppleScript seems necessary for screenshot/UX work, first treat that as a RegionShot capability gap: report the use case and suggest the RegionShot improvement before using the fallback.
- RegionShot is maintained by this project; if it has a bug or missing capability, report the concrete use case and suggest a tool improvement instead of silently accepting the limitation.
